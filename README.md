# Particle-physics-paper-datasets

本リポジトリは、F2DV / GAV / RQT_v2.1 の各プロジェクトに関連するデータと補助コードをまとめたものです。内容は研究・再現性のための素材であり、READMEの記述は結果や解釈を固定するものではありません。
This repository aggregates data and auxiliary code related to the F2DV / GAV / RQT_v2.1 projects. The contents are materials for research and reproducibility, and the README does not fix or constrain results or interpretations.

## 構成 / Structure
- F2DV: データ、図、解析ノートブック
  F2DV: data, figures, analysis notebooks
- GAV: データ、図、補助資料
  GAV: data, figures, supporting materials
- RQT_v2.1: データ、解析ノートブック
  RQT_v2.1: data, analysis notebooks

## 取り扱い / Usage
ファイル構成は変更される可能性があります。パスやファイル名に依存する場合は、必要に応じて更新してください。
The file structure may change. If you rely on paths or filenames, please update them as needed.

データや図表はそのままの形で保存されており、READMEは更新や変更の影響を与えません。
Data and figures are stored as-is, and the README does not affect updates or changes.
